Bilingual (Polish/English) Multiple Intelligences survey based on Howard Gardner's theory.

== Description ==

MI Survey delivers a bilingual Multiple Intelligences survey. It collects user responses, scores them against 8 intelligence types, generates a bar chart profile, provides written descriptions, and allows the user to download or email the report.

== Installation ==

1. Upload the `mi-survey` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode `[mi_survey]` on any page or post
